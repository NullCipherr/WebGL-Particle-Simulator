# WebGL Particle Simulator

Projeto frontend em React + Vite para simulacao de particulas com WebGL.

## Estrutura

```text
.
├── .github/workflows/      # CI de validacao (typecheck + build)
├── docs/                   # Documentacao operacional
├── public/                 # Assets e arquivos estaticos do Pages
├── src/
│   ├── components/
│   ├── constants/
│   ├── engine/
│   └── types/
├── .env.example
├── index.html
├── package.json
├── tsconfig.json
├── vite.config.ts
└── wrangler.toml
```

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
